| **层级**        | **负责内容**         | **与你的 MongoDB 关联**           |
| ------------- | ---------------- | ---------------------------- |
| **Data**      | 原始行情、财务报表        | 维护那 **14M** 条 `market_daily` |
| **Factor**    | 动量、IC/Rank IC 计算 | 基于 `ts_code` 索引飞速提取因子        |
| **Model**     | 机器学习/多因子回归       | 训练并保存 `.pkl` 模型文件            |
| **Strategy**  | 组合选择、信号过滤        | 决定具体的 Portfolio 权重           |
| **Risk**      | 止损、仓位控制          | 确保 850 万条历史回测中不爆仓            |
| **Execution** | 撮合、手续费计算         | 还原真实的交易成本                    |
| **Backtest**  | 时间轴滚动、引擎循环       | 驱动整个流程运行                     |
| **Evaluator** | 统计分析、图表展示        | 告诉你这个策略到底能不能挣钱               |
| **Util**      | 配置中心、日志、通知       | 发送钉钉/微信预警到你的手机               |
