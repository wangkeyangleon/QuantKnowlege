## 使用 machine learning
### 本质
使用机器学习只是基于历史的来发觉出来股票价格的规律，其实本质就是历史会进行重演，历史可以代表未来
### 缺点：
历史并不能代表未来
### 提升点
只是作为一个辅助，在进行推荐股票的时候作为一个参考值，最终目标应该是通过因子+机器学习+ai的分析三者进行加权得到买卖建议

| **模块**                    | **作用**       | **对应量化概念** |
| ------------------------- | ------------ | ---------- |
| **Indicator Calculation** | 生成 X（特征）     | 因子挖掘       |
| **Labeling**              | 生成 y（目标）     | 标注数据       |
| **XGBoost**               | 拟合 X 与 y 的关系 | 策略建模       |
| **Backtesting**           | 计算收益         | 策略评价       |
### 程序功能
- 读取本地 股票数据。
- 自动算好几十个技术指标。
- 训练一个 XGBoost 森林模型。
- 告诉你今天该买哪几个 ETF。
    
- 画出一张图，告诉你如果过去几年按这个跑，能赚多少钱。
