## 如何避免追高买入
### 设定买入原则
- 不在涨停板、连续大阳线后盲目买入
- 坚持只在支撑位、回调企稳时减仓
### 看成交量与形态
- 放量滞涨、长上影线要警惕
- 等待缩量回踩再进，而不是追涨加速阶段
### 分批建仓
- 不一次性满仓，分两三次布局，降低买在最高点的风险。
### 设定冷静期
- 出现热门概念、媒体铺天盖地报道时，给自己设一个24小时冷静期，避免冲动

### 如果已经追高被套，如何解套
1、先判断趋势
- 如果只是短期调整，趋势依旧向上，可以耐心等待反弹
- 如果明显见顶转弱，不能死扛，要考虑止损或换股
2、减仓换股法
- 卖出一部分套牢股，腾出资金去做强势股回血
3、地位补仓摊成本
- 在合理只层位小仓位补仓，降低持仓成本
4、止损认错法
- 如果买入逻辑彻底破坏，果断割肉，保存实力

## 股票类型
### 价值股
价格低于内在价值的股票，通常估值便宜、偏稳健，常见于银行、保险等板块。
- **市盈率（PE）**：反映市场愿意为公司每一元净利润支付的价格。他代表了按当前盈利水平，回本所需的年数。股价 / 每股收益（EPS）
- **市净率（PB）**：反映市场愿意为公司每一元账面净资产支付的价格。账面净资产即股东权益（总资产减去总负债）。股价 / 每股净资产
- 股息率： 每股股息 / 股价
- PEG：修正后的市盈率，用于弥补PE忽略成长性的缺陷。它衡量的是在考虑公司利润增速的前提下，当前的估值是否合理。 PEG = 市盈率 / 净利润增长率
代表：银行、保险
### 成长股
业绩和营收保持高速増长的股票，通常估值较高，代表如能源、科技公司

### 白马股
业绩好、知名度高、信息透明度强的公司股票，走势稳健，常被机构长期持有
代表：茅台、伊利股份

### 蓝筹股
行业龙头、大市值、盈利稳定的股票，通常有分红，是指数的核心构成部分。
代表：中国石油、中国建筑
### 红筹股
在境外（如港股）上市的、具有中国内地背景的公司股票，注册地在海外，主要业务在中国。
代表：中国移动、华润啤酒


### 题材股
围绕热点概念炒作的股票，具有很大想象空间，但业绩不一定兑现，风险较高

### 周期股
业绩随经济周期波动的股票，景气时盈利丰厚，低估时可能亏损，如钢铁、有色、煤炭。

### 防御股
受经济波动影响小、现金流稳定的股票，典型行业有公用事业、医药、消费

### ST股
经营或财务状况异常的股票，被交易所特别处理，风险较高，存在退市可能

### 科技股
以技术创新驱动成长的股票，成长性强，但波动大，对政策或市场环境敏感。



## 三周期选股
### 大周期看方向（是否值得参与）
- 判断股价处于长期上升趋势中
- 回避熊市、平台震荡或下跌趋势中的票
- 常用指标：月线均线多头排列、周线上升通道
- 示例：若月线MACD刚刚金叉，或周线站稳MA60，可视为“方向正确”

### 中周期看结构（是否处于低吸位置）
- 分析个股在日线层面处于哪一波动阶段（启动/调整/突破）
- 寻找日线级别的回调结束点或突破点
- 常用形态：箱体震荡、趋势回踩、W底、旗形整理等
- 示例：个股回踩日线20日均线后止跌回升，是低吸区间

### 小周期抓时机（择时进出）
- 在大/中周期条件成熟的前提下，使用小周期确认买点
- 等待60分钟级别的放量突破、回踩确认、底背离等信号
- 小周期也决定你是否追高、错过、或低吸成功

### 实战流程
- 筛选大周期向上的板块与个股
- 在日线中寻找整理后的突破形态（如：箱体突破、上升旗型、均线支撑上移）
- 用60分钟图确认入场时机（如：缩量回踩后企稳、放量上穿均线、MACD金叉）

### 总结
大周期定方向、中周期选结构、小周期抓节奏。三者共振出黄金点，趋势反转避其锋。

## 回调后选股

#### 选股逻辑 
1、系统性回调（市场整体下跌）
- 多数个股都会跌，这是宏观流动性或市场情绪导致
- 好公司会被错杀，出现价值洼地
2、结构性回调（行业或个股调整）
- 可能是景气周期波动、政策扰动、结构性估值消化
- 若长期逻辑围边，则是捡筹码的良机
3、根本性回调（基本面恶化）
- 企业业绩下滑、赛道前景逆转、竞争格局恶化
- 这类下跌不是机会，而是风险

#### 选股框架
1、看基本面（把握长期确定性）
- 行业：是否是长期高景气赛道？
- 公司：是否具备护城河（龙头地位、核心资源、技术壁垒）
- 财务：盈利是否稳健、负债是否安全、现金流是否充足
2、看估值（是否跌到合理区间）
- 和历史估值对比（PE、PB、PS等）
- 和同类龙头对比
- 是否跌到安全边际，未来盈利能否支撑当前估值
3、看资金面（谁在卖，谁在买）
- 主力资金是否止跌回流
- 外资、机构是否在加仓
- ETF、产业资本是否有抄底迹象
4、看技术面（用作辅助择时）
- 是否出现止跌企稳信号（量缩价稳、放量反弹）
- 是否回踩长期均线支撑
- 是否走出底部形态

## 趋势选股的买入方法
### 买入趋势股的关键
- 选择具备上涨逻辑的股票
- 在趋势启动或回调结束时买入
- 结合量价关系、形态突破和资金动向，判断买入时机

### 基本面筛选 -- 选择具备上涨逻辑的个股
- 行业景气度上升
- 公司业绩持续增长
- 政策利好支持
- 资金持续流入

### 技术面筛选 -- 判断趋势的启动信号
- 突破平台或长期压力位
	- 突破箱体整理或关键均线（如60日、120均线）
	- 放量突破，确认多头趋势形成
- 均线多头排列
	- 5日线、10日线、20日线、60日线呈现多头排列
	- 股价沿着均线向上，均线形成明显的上升趋势
- 量价配合良好
	- 上涨放量，下跌缩量 --》资金在积极推动
	- 主升浪中成交量逐步方法，表明资金接入
- MACD、KDJ等指标金叉
	- MACD红柱放大，零轴上方金叉
	- KDJ超跌金叉，提示趋势反转
- 突破后回踩不破，再度拉升
	- 突破后回踩确认支撑有效，是低吸的好机会

### 买入时机
- 突破买入法（趋势启动）
	- 放量突破关键压力位
	- 回踩确认支撑后，缩量回踩时低吸
	-例：股价长期在60日线下方整理
		放量突破60日线
		回踩60日线后获得支撑 ->果断买入
- 回调买入法（趋势延续）
	- 趋势股在上涨途中，通常会有技术性回调
	- 逢回调至关键支撑位或20日线、60日线附近低吸
	-例：股价上涨->短期回调至20日线->回踩企稳
		企稳后，量能恢复->低吸
- 整理突破买入法（趋势加速）
	- 股价在平台整理中，成交量逐渐萎缩
	- 放量突破后，突破收盘站稳->加速上行信号
	-例：突破区间震荡箱体
		放量突破站稳 ->顺势加仓

### 移动止盈策略
- 股价沿趋势上涨，设置移动止盈位，防止利润回吐
#### 常用止盈方法
- 5日线止盈法：突破主升浪，沿5日线持股
- ATR（平均波幅）止盈法：设置一定波动幅度，触及则止盈
	- 示例：突破回踩不破5日线 ->继续持有
			失守五日线 ->移动止盈，部分止盈

# **方法一：计算均线斜率（★★★★★ 推荐）**

例如月线MA5：

```
MA5 = 最近5个月收盘价平均

Slope = (MA5_today - MA5_3个月前) / MA5_3个月前
```

例如：

|**月份**|**MA5**|
|---|---|
|3个月前|10.0|
|现在|10.1|

那么

```
Slope = (10.1-10)/10

=1%
```

说明：

MA5三个月只涨了1%

基本就是横着走。

---

可以定义：

```
abs(Slope)<3%
```

认为

```
MA5开始走平
```

MA10同理。

例如

```
abs(Slope5)<3%

AND

abs(Slope10)<2%
```

满足即可。

## **我建议再升级一步：识别”筑底完成”而不是仅”走平”**

从你最近一直在做的 **Mistery 趋势量化系统** 来看，单纯判断均线走平还不够，因为很多股票在长期下跌过程中也会短暂走平。

因此，我建议把这个模块升级成 **“月线筑底识别器”**，至少包含以下 6 个条件：

| **模块**     | **量化条件**                  | **建议权重** |
| ---------- | ------------------------- | -------- |
| 12个月平台宽度   | `(HH12-LL12)/LL12 < 25%`  | 20%      |
| MA5斜率      | `abs(Slope5) < 3%`        | 20%      |
| MA10斜率     | `abs(Slope10) < 2%`       | 20%      |
| MA5与MA10距离 | `abs(MA5-MA10)/MA10 < 2%` | 15%      |
| 月ATR收缩     | ATR/Close 持续下降且低于阈值       | 15%      |
| 月成交量收缩     | 近6个月平均成交量低于近24个月平均        | 10%      |

当综合得分达到 **80 分以上** 时，就可以认为股票进入了高质量的月线筑底阶段。这种设计比单独依赖”均线走平”更稳健，也更符合Mistery强调的”长时间平台整理、波动收敛，然后突破”的思想。

我觉得，这个模块可以直接作为你 **AiQuant V10** 的核心选股模块之一，后面再叠加周线突破、日线放量启动等条件，就能形成一套完整的 **“月线筑底 → 周线突破 → 日线启动”** 多周期趋势交易系统。
